# API BookStore

### Create .env
```.env
USER_DB=<postgres>
PASS_DB=<password>
HOST_DB=<localhost>
NAME_DB=bookstore
PORT_DB=<5001>

PASS_SECRET=<password_secret_token>
```

### Exec
```npm
npm install -g typescript
npm i
npm run dev
```
